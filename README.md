# Triage Gamer

Proyecto web simple con dos interfaces: clínica y versión "gamer".

## Descripción
Este repositorio contiene los archivos estáticos del proyecto (HTML, CSS, JS, imágenes) para la demo de Triage.

## Uso
- Abrir `index.html` para la versión clínica.
- Abrir `index_gamer.html` para la versión gamer.

Puedes servirlo con cualquier servidor estático o abrir los archivos directamente en el navegador.

Ejemplo con Python (desde la carpeta del proyecto):

```powershell
python -m http.server 8000
```

Abrir http://localhost:8000 en el navegador.

## Estructura
- `index.html`, `index_gamer.html` — páginas principales
- `css/` — estilos
- `js/` — scripts
- `images/` — recursos gráficos
